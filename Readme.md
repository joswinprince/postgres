
# Postgres

### List of Commands
***
>**Pullout version**
```
SELECT version();
```
>**Print date**
```
SELECT now();
```
>**Creating the database in POSTGRES SQL**
```
CREATE DATABASE name;
```
>**List all the database in the POSTGRES SQL**
```
\l
```
>**Switching the database** 
```
\c databaseName
```
>**Create a table** 
```
create table colors(colorId int,colorName char(20));
```
